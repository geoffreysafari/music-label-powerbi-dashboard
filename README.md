# music-label-powerbi-dashboard
Power BI dashboard analyzing music label revenue, royalties, and performance

## 📊 Music Label Power BI Dashboard

This project analyzes a simulated music label dataset using Power BI.

### 🔹 Features:
- Interactive dashboards with slicers
- Revenue analysis across artists, promoters, and producers
- Dynamic filtering by year and genre

### 🔹 Tools Used:
- Power BI
- Data Modeling
- Data Visualization

### 🔹 Key Insights:
- Identified top-performing artists and songs
- Analyzed revenue distribution across stakeholders
- Highlighted impact of high-performing songs
